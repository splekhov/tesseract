This project created becaue of attempt to find out a way to search in huge amount of screenshots on my mac for plain text inside:  
./find_text_in_images_tesseract.sh ./funny-staff/ "zwei"  
Scanning images in: ./funny-staff/  
Looking for text: zwei  

MATCH: ./funny-staff/zwei.png  

P.S. prerequisites: brew install tessaract  
P.P.S. doesn't work with Russian text (todo)
